  2. 문해력 코딩이란?

시나리오 기반 문제 분석 → 알고리즘 설계 → 프로젝트 구현

교육 핵심 3축

Storytelling · G‑PBL · STEM을 결합해 창의적 융합 사고력을 키웁니다.

Storytelling

쉽게 접할 수 있는 이야기 속에서 문제를 발견하고, 논리적 스토리 분석과 마인드맵으로 사고를 구조화합니다.

G‑PBL

팀 프로젝트 기반으로 브레인스토밍 → 설계 → 구현 → 발표까지 경험하며 협업과 소통 역량을 키웁니다.

STEM

사회적·현실적 문제를 수학·과학적 사고로 연결해 융합적 관점에서 해결하는 힘을 기릅니다.

Class Procedure

한 수업 안에서 “이해 → 실습 → 프로젝트”까지 이어집니다.

  • Phase 1 — 당일 수업 내용에 대한 기본 설명
  • Phase 2 — IT 주제 영상 시청(옵션)
  • Phase 3 — 주제 기반 Storytelling & 문제 정의
  • Phase 4 — 알고리즘 설계 및 코딩 실습
  • Phase 5 — 프로젝트 수행 및 결과 공유

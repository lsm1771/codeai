Class Procedure

한 수업 안에서 “이해 → 실습 → 프로젝트”까지 이어집니다.

  • Phase 1 — 당일 수업 내용에 대한 기본 설명
  • Phase 2 — IT 주제 영상 시청(옵션)
  • Phase 3 — 주제 기반 Storytelling & 문제 정의
  • Phase 4 — 알고리즘 설계 및 코딩 실습
  • Phase 5 — 프로젝트 수행 및 결과 공유

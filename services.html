시나리오(글)를 읽고 핵심 요소를 추출해 문제를 정의한 뒤, 논리적으로 설계하고 프로젝트로 구현하는 수업입니다.

온라인 코드 협업 환경

수업은 ‘줌 화면공유’로 끝나지 않아요. 실제 개발팀처럼 협업 도구를 쓰며 함께 완성합니다.

실시간 라이브 코딩

Zoom + VS Code Live Share/브라우저 IDE로 같은 화면을 보며 ‘같이 코딩’합니다.

  • 즉시 디버깅
  • 코드 설명/질문 루프
  • 녹화/요약(선택)

GitHub 협업 워크플로우

브랜치 → PR → 리뷰 → 머지까지. 협업 개발의 표준 루틴을 그대로 적용합니다.

  • 브랜치 전략
  • PR 리뷰 코멘트
  • 이슈/칸반 관리

프로젝트 그룹 운영

4~8인 그룹은 스프린트 방식으로 운영합니다. 역할 분담과 데모데이 발표로 ‘완성 경험’을 만듭니다.

  • 역할(PM/Dev/QA) 분담
  • 주간 스프린트 회고
  • 결과물 포트폴리오화

시각자료로 보는 문해력 코딩

이미지를 클릭하면 확대해서 볼 수 있습니다.

Storytelling 쉽게 접할 수 있는 이야기 속에서 문제를 발견하고 논리적으로 분석합니다.
STEM 융합 사회적 주제를 수·과학과 접목해 ‘융합적 사고’를 확장합니다.

수업의 핵심 구조

스토리텔링(읽기) → 문제 분석(요소추출) → 프로젝트 구현(STEM 확장)

Storytelling

시나리오를 읽고 핵심 단어/조건을 표시하며 문제의 본질을 찾습니다.

논리적 문제 분석

요소를 추출하고 마인드맵/순서도로 정리해 해결 절차(알고리즘)를 설계합니다.

STEM 프로젝트 구현

수학·과학·공학 요소를 접목해 코딩으로 구현하고, 결과를 검증·개선합니다.

수업 진행 흐름

01

시나리오 읽기

이야기/지문을 읽고 상황을 이해합니다.

02

요소 추출

핵심 단어/조건/목표를 표시하고 정리합니다.

03

논리 구조화

마인드맵/순서도로 해결 절차를 설계합니다.

04

코딩 구현

설계한 절차를 코드로 구현합니다.

05

검증 & 개선

결과를 검증하고 피드백을 반영해 완성도를 높입니다.

수업 상담이 필요하신가요?

학생의 수준/목표에 맞게 과정과 수업 흐름을 안내드립니다.

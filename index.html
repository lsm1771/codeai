왜 문해력 기반 코딩이 필요한가요?

코딩은 ‘컴퓨터가 이해하는 언어’로 논리적 글을 써 내려가는 과정입니다. 코드아이는 문장 이해 → 논리 설계 → 구현 → 검증을 반복하며, 단순 문법 암기가 아닌 사고력 중심의 학습을 만듭니다.

  • 01 요구사항을 “문장”으로 정리하면 구현이 쉬워집니다

    요구사항을 문장으로 정리하고, 조건/예외를 체크한 뒤 구현합니다. 실수가 줄고 디버깅이 쉬워집니다.

  • 02 그룹 프로젝트로 ‘협업 코딩’ 역량을 키웁니다

    역할 분담, 일정, 리뷰를 경험하면서 ‘혼자 잘하는 코딩’에서 ‘함께 완성하는 코딩’으로 성장합니다.

  • 03 STEM 융합으로 ‘작동하는 결과물’을 만듭니다

    센서/데이터/모델을 활용해 ‘작동하는 결과물’을 만드는 경험이 진짜 실력을 만듭니다.

클릭하면 확대해서 볼 수 있어요.

Class Procedure

한 번의 수업이 “체험”으로 끝나지 않게, 매 시간 구조화된 루틴으로 학습을 완성합니다.

Phase 1

주제 몰입

IT/사회/과학 주제에 대한 영상·자료를 보고 핵심을 요약합니다.

Phase 2

문해력 기반 논리 분석

조건·규칙·예외를 문장으로 정리하고, 의사코드/흐름도로 설계합니다.

Phase 3

코딩 구현

개념을 실제 코드로 구현하고, 역할 분담으로 프로젝트를 진행합니다.

Phase 4

테스트 & 디버깅

오류를 발견하고 원인을 찾는 ‘디버깅 루틴’을 학습합니다.

Phase 5

발표 & 피드백

결과물을 공유하고 개선 포인트를 정리하여 다음 단계로 확장합니다.

결과물이 남는 수업

매 수업 “기획서 → 구현 → 테스트 기록”이 남아 포트폴리오로 확장됩니다.
